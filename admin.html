Información del Sistema

Detalles sobre cómo funciona este panel de administración.

¿Cómo funciona?
La configuración de disponibilidad se guarda localmente en este navegador (localStorage). Los clientes que visiten el sitio web desde el mismo servidor/host verán los horarios que configures aquí.

Para múltiples dispositivos: Exporta la configuración como JSON y luego impórtala en el otro dispositivo usando la función "Importar Configuración".

Contacto técnico: Si necesitas que la disponibilidad se sincronice en la nube para múltiples dispositivos, se requiere una actualización con base de datos (Supabase o similar).
